# Number Guessing Game

A simple and interactive command-line game built using Python, where the player tries to guess a randomly generated number within a limited number of attempts.

---

## 🚀 Features

* Custom input range (lower and upper bounds)
* Random number generation
* Limited attempts to guess the number
* Helpful hints ("Too High" / "Too Low")
* Beginner-friendly Python project

---

## 🛠️ Tech Stack

* Python
* Built-in `random` module
